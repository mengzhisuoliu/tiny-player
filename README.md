# Tiny Player

## 简介

Tiny Player 是一个轻量、可扩展的 HTML5 视频播放器，支持 mp4、webm、ogg 等多种常见格式，支持 m3u8，支持自动切换。

- 不依赖任何框架，任何浏览器都可以使用，支持移动端。
- 支持自定义控制栏挂载目标节点，支持自定义控制栏组件显示隐藏。
- 仅 10kb 大小，gzip 压缩后仅 4kb 大小。
